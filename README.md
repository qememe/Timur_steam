# Тимур Steam Библиотека

Библиотека игр с открытым исходным кодом, написанная на C++ и Qt 6.

## Особенности
- **Библиотека игр**: Просмотр и управление коллекцией веб-игр.
- **Интеграция с GitHub**: Игры скачиваются (клонируются) напрямую с GitHub репозиториев.
- **Умный запуск**: Автоматическое обнаружение HTML файла запуска (поддержка нестандартных имен файлов).
- **Надежная установка**: Автоматическое восстановление при поврежденных загрузках.
- **Встроенный браузер**: Запуск игр прямо внутри приложения.
- **Режим разработчика**: Возможность добавлять новые игры в локальный каталог (генерирует JSON для публикации).

## Требования для сборки

### Автоматическая сборка (Рекомендуется)

#### Linux
Запустите скрипт сборки (автоматически установит зависимости и соберет проект):
```bash
./bild.sh
```

#### Windows
Запустите файл `bild.bat`.

---

### Ручная сборка

#### Linux (Arch Linux)
1. Установите зависимости:
   ```bash
   sudo pacman -S base-devel cmake git qt6-base qt6-webengine
   ```
2. Сборка:
   ```bash
   mkdir build && cd build
   cmake ..
   make
   ```
3. Запуск:
   ```bash
   ./Timur_steam
   ```

#### Windows
1. Установите [Qt 6](https://www.qt.io/download) (включая WebEngine).
2. Установите [CMake](https://cmake.org/download/).
3. Установите [Git](https://git-scm.com/download/win).
4. Откройте проект в Qt Creator или используйте командную строку:
   ```cmd
   mkdir build
   cd build
   cmake ..
   cmake --build . --config Release
   ```

## Как использовать
1. **Библиотека**: Выберите игру из списка. Если она не установлена, нажмите "УСТАНОВИТЬ" (скачивание через Git).
2. **Играть**: После установки дважды кликните по игре, чтобы запустить её (кнопка "ИГРАТЬ").
3. **Разработчик**:
   - Нажмите "Вход" (введите любое имя).
   - Нажмите "Стать разработчиком".
   - Появится кнопка "Добавить игру (Dev)".
   - Заполните форму (ID, название, URL репозитория).
   - Игра добавится в ваш локальный список.
